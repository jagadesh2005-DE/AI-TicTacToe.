# AI Tic Tac Toe

This repository contains implementations of the Tic Tac Toe game using Artificial Intelligence algorithms.

## Algorithms Used

1. **Minimax Algorithm**
2. **Alpha-Beta Pruning**

These algorithms allow the computer to play optimally against a human player.

## Files in this Repository

* `minimax_tictactoe.py` – Tic Tac Toe implemented using the Minimax algorithm.
* `alphabeta_tictactoe.py` – Tic Tac Toe implemented using Alpha-Beta pruning optimization.

## Requirements

* Python 3.x

## How to Run

Run the programs using Python:

```
python minimax_tictactoe.py
```

or

```
python alphabeta_tictactoe.py
```

The user plays with **X** and the computer plays with **O**.
Enter a position from **1–9** to place your move on the board.

## Example Board Positions

```
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
```

## Output

The program displays the Tic Tac Toe board after every move and announces whether the **user wins, computer wins, or the game ends in a draw**.
